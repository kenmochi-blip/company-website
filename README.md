# Smart & Smooth - 会計研修サービス ランディングページ

![Smart & Smooth](https://page.gensparksite.com/v1/base64_upload/5b6d92b0c9b16a34bd828b5afe15dff1)

## 🎯 プロジェクト概要

**Smart & Smooth** は、元・上場企業副社長で公認会計士の剱持 健氏が代表を務める会計研修サービスのランディングページです。「会計アレルギーを経営の武器に変える」というビジョンのもと、監査法人（理論）とIPO実現（実践）の経験を活かし、企業の「数字に強い人材」を育成する研修プログラムを提供しています。

### ブランドカラー
- **ライトブルー**: #00D4FF
- **ディープブルー**: #0052FF
- **グラデーション**: linear-gradient(135deg, #00D4FF 0%, #0052FF 100%)

## ✨ 実装済み機能

### 1. **ファーストビュー（Hero Section）**
- キャッチコピーとサブコピーの強力な訴求
- ブランドカラーのグラデーション背景
- 4つの権威付け要素（PwC出身、元副社長、研修実績、認定インストラクター）
- 2つのCTAボタン（無料相談・資料ダウンロード）
- スクロールダウンアニメーション

### 2. **課題提起セクション（Problems Section）**
- 3つの顧客セグメント別の課題提示
  - 管理職・マネージャー層
  - IPO準備・経営層
  - 過去の研修での失敗
- チェックマーク付き課題リスト
- 解決への導線（グラデーションボックス）

### 3. **選ばれる3つの理由（Strengths Section）**
- **実践力**: IPO実現の当事者としての経験
- **教育力**: エデュテイナーとしての「楽しく学ぶ」アプローチ
- **信頼性**: PwC出身公認会計士としての専門性
- 各理由にナンバリングとアイコン付き
- 左右交互配置のダイナミックレイアウト

### 4. **研修プログラム（Programs Section）**
#### プログラムA: 戦略MGマネジメントゲーム® 研修
- 体感・意識改革コース
- ゲーム形式での経営体験
- 受講者の声（2件）

#### プログラムB: 会計基礎・分析 実践研修【人気コース】
- 実践・知識習得コース
- YouTube動画埋め込み（カフェ経営シミュレーション）
- 大手・上場製薬会社での実績紹介
- おすすめ企業の明確化

#### カスタマイズ対応
- 柔軟なプログラム設計の案内

### 5. **信頼と実績（Trust Section）**
- **企業研修実績**: 大手・上場製薬会社などの具体例
- **第三者の声**: PwC Japan パートナーからの推薦コメント

### 6. **講師プロフィール（Profile Section）**
- 剱持 健氏の詳細な経歴
- プロフィール写真の効果的な配置
- ビジョンの強調表示
- 資格・実績の視覚的な表示（4つのクレデンシャル）

### 7. **お問い合わせ（Contact Section）**
- 最終CTA（2つのボタン）
- サポート情報（オンライン/対面、カスタマイズ対応）
- お問い合わせフォーム
  - 会社名、お名前（必須）
  - メールアドレス（必須）
  - 電話番号
  - ご興味のあるプログラム（選択式）
  - 想定受講者数
  - お問い合わせ内容（必須）
  - 個人情報の取り扱い同意チェックボックス
- リアルタイムバリデーション

### 8. **共通機能**
- **レスポンシブデザイン**: PC/タブレット/スマートフォン完全対応
- **固定ヘッダー**: スクロール時も常に表示
- **モバイルメニュー**: ハンバーガーメニュー対応
- **スムーズスクロール**: アンカーリンクのスムーズ遷移
- **スクロールアニメーション**: 各セクションのフェードイン効果
- **トップへ戻るボタン**: スクロール時に表示
- **フォーム送信処理**: バリデーション＋成功通知
- **通知システム**: 成功/エラーメッセージの表示

## 🔧 カスタマイズガイド

### ブランドカラーの変更
`css/style.css` の `:root` セクションで色を変更できます：

```css
:root {
    --primary-light: #00D4FF;    /* ライトブルー */
    --primary-deep: #0052FF;     /* ディープブルー */
    --primary-gradient: linear-gradient(135deg, #00D4FF 0%, #0052FF 100%);
}
```

### コンテンツの編集
`index.html` で各セクションのテキストを直接編集できます：
- セクション1: `.hero` (行22〜)
- セクション2: `.problems` (行78〜)
- セクション3: `.strengths` (行124〜)
- セクション4: `.programs` (行224〜)
- セクション5: `.trust` (行380〜)
- セクション6: `.profile` (行470〜)
- セクション7: `.contact` (行525〜)

### 画像の変更
画像URLを変更する場合：
```html
<!-- ロゴ -->
<img src="YOUR_LOGO_URL" alt="Smart & Smooth Logo">

<!-- プロフィール写真 -->
<img src="YOUR_PROFILE_IMAGE_URL" alt="剱持 健">
```
