# 大人の寺子屋「歴読ウォーク」公式サイト

完全紹介制・歴史探訪コミュニティの公式ウェブサイト

## 🎯 プロジェクト概要

「大人の寺子屋『歴読ウォーク』」は、歴史や日本文化をテーマに、「インプット（読書）」と「アウトプット（現地探訪）」を組み合わせた会員制コミュニティです。

### コンセプト
- **キャッチコピー**: 「日本」を学びのフィールドに、深く繋がり、共に育つ「最高の居場所」
- **サブコピー**: 本で学び、現地で感じる。大人のための会員制・歴史探訪コミュニティ
- **特徴**: 完全紹介制、最大20名限定のクローズドコミュニティ

## 🔧 カスタマイズガイド

### 1. メールアドレスの変更
`index.html` のContactセクション（line 560付近）を編集：

```html
<a href="mailto:info@smartandsmooth.com" class="contact-link">
    info@smartandsmooth.com
</a>
```

### 2. 共同代表情報の追加
`index.html` のOrganizerセクション（line 480付近）を編集：

```html
<div class="organizer-role">共同代表</div>
<h3 class="organizer-name">〇〇 〇〇</h3>
<div class="organizer-bio">
    <p>プロフィール文...</p>
</div>
```

### 3. 会員規約PDFのリンク設定
規約PDFを作成後、`index.html` のFooterセクションを編集：

```html
<a href="pdf/membership-rules.pdf" class="footer-doc-link" target="_blank">
    <i class="fas fa-file-alt"></i>
    会員規約（PDF）
</a>
```

### 4. カラーの変更
`css/style.css` の `:root` セクションでカラー変数を変更：

```css
:root {
    --primary-color: #1B3D6D;      /* メインカラー */
    --secondary-color: #C8A882;    /* アクセントカラー */
}
```
